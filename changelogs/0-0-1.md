# Changelog

All notable changes to this workspace are documented here. Versions follow the monorepo **app** `package.json` version when we cut releases; until then entries stay under **[Unreleased]**.

## [0.0.1]

### Added

- **`@galipette/database`** — Prisma 7 PostgreSQL package: `Character`, `CharacterSheet`, `prisma.config.ts`, driver adapter (`@prisma/adapter-pg` + `pg`), initial migration under `prisma/migrations`, singleton `PrismaClient`, package README.
- **`@galipette/shared-schemas`** — Zod DTOs for character flows plus re-exported Prisma types (`Character`, `CharacterSheet`, `Prisma`); package README.
- **`docs/database-cheatsheet.md`** — Operational guide (connection string, migrate, deploy, push, Studio, raw SQL, new tables, manual data).
- Root **`package.json`** scripts: `build:database`, `build:shared-schemas`, and **`db:*`** forwards (`db:generate`, `db:validate`, `db:migrate`, `db:deploy`, `db:push`, `db:studio`, `db:status`).
- **`db:status`** on **`@galipette/database`** — `prisma migrate status`.

### Changed

- **Root README** — Quick links and commands for database, shared-schemas, cheat sheet, and changelog; Contributing points to DB docs and this file.

### Other

- **`.gitignore`** — `packages/database/src/generated` (Prisma `prisma-client` output).
- **`pnpm-workspace.yaml`** — `allowBuilds` for Prisma packages set to `true` so install scripts run.

### Dependencies (database stack)

- Prisma ORM **7.x** with **`prisma-client`** generator (output under `packages/database/src/generated/prisma`), **`dotenv`**, **`pg`**, **`@prisma/adapter-pg`**.
